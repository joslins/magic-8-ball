# Magic 8 Ball 🎱

A kid-friendly iOS app built with Swift and Xcode. Shake your phone to ask the Magic 8 Ball a yes/no question and get a mystical answer.

## Features

- Shake detection — shake your phone to reveal an answer
- Smooth animations and transitions
- Sound effects and haptic feedback
- Custom splash screen
- 20 classic Magic 8 Ball responses

## Tech stack

- Swift
- Xcode
- UIKit
- Core Motion (shake detection)
- AVFoundation (sound effects)

## Getting started

1. Clone the repo
2. Open `Magic 8 Ball.xcodeproj` in Xcode
3. Select a simulator or connected iPhone
4. Hit Run (Cmd+R)
